Program FizzBuzz!

Enter a number and "FizzBuzz!".

Rules:

  1. if number is divisible by 3, print "Fizz".
  2. if number is divisible by 5, print "Buzz".
  3. if number is divisible by both 3 and 5, print "FizzBuzz".
  4. else just print the number
